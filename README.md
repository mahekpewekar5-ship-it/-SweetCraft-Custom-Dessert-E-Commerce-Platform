SweetCraft is a full-stack e-commerce application built using Spring Boot and Angular, focused exclusively on customizable desserts. Users can browse products like cakes, waffles, pancakes, pastries, cupcakes, and bento cakes, customize them with specific preferences and messages, add items to the cart, place orders, and track order status. The admin panel allows order management, viewing customization details, and updating order progress efficiently.
